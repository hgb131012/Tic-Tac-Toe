# Tic Tac Toe 🎮

A classic two-player Tic Tac Toe game built with HTML, CSS, and JavaScript. This project is part of [The Odin Project](https://theodinproject.com) curriculum to practice DOM manipulation and factory functions/module patterns.

## 🚀 Live Demo
You can play the game directly in your browser here: **[Tic Tac Toe Game](https://hgb131012.github.io/Tic-Tac-Toe/)**

## ✨ Features
- **Two-player mode**: Play locally with a friend (X and O).
- **Interactive UI**: Clean, modern design with smooth animations.
- **Game State Management**: Automatically detects wins, losses, and ties.
- **Reset functionality**: Start a new game instantly without refreshing the page.

## 🧠 Things Learned
Through this project, I strengthened my understanding of:
- **Factory Functions and Modules**: Encapsulating game logic to keep the global scope clean.
- **DOM Manipulation**: Updating the UI dynamically based on the JavaScript game state.
- **Event Listeners**: Handling user clicks and resetting the game board efficiently.
